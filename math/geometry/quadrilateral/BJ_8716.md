> Question

* Print overlapped area of two rectangle

```txt
Input:
0 3 4 0
2 4 6 1

Output: 4
```

{% endtab %}
{% tab title='BJ_8716.py' %}

```py
x1, y1, x2, y2 = map(int, input().split())
x3, y3, x4, y4 = map(int, input().split())
Xu, Yu = max(x1, x3), min(y1, y3)
Xl, Yl = min(x2, x4), max(y2, y4)
print(max(0, Xl - Xu) * max(0, Yu - Yl))
```
