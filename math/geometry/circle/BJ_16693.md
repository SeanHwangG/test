> Question

* 첫 줄에 피자의 넓이와 가격
* 두번 째 줄에 피자의 반지름과 가격이 주어질 때
* 두번째 줄이 싸면 Whole pizza, 아니면 Slice of pizza를 출력하라

{% endtab %}
{% tab title='BJ_16693.py' %}

```py
import math
a, p1 = map(int, input().split())
r, p2 = map(int, input().split())
if a * p2 < r * r * math.pi * p1:
  print('Whole pizza')
else:
  print('Slice of pizza')
```
