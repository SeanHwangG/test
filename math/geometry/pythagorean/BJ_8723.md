> Question

* Given three sides print 2 if equilateral, 1 if right triangle, else 0

```txt
Input: 3 4 5
Output: 1
```

{% endtab %}
{% tab title='BJ_8723.py' %}

```py
a, b, c = sorted(map(int, input().split()))
if a == b and b == c: print(2)
elif a ** 2 + b **2 == c**2: print(1)
else: print(0)
```
