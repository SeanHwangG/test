> Question

* Given width and height of rectangle print width + height - diagonal

```txt
Input: 3 4
Output: 2
```

{% endtab %}
{% tab title='BJ_20215.py' %}

```py
w, h = map(int, input().split())
print(w + h - (w ** 2 + h ** 2) ** 0.5)
```
