> Question

* Given three angels, print triangle type

```txt
Input:
60
70
50

Output: Scalene
```

{% endtab %}
{% tab title='BJ_10101.py' %}

```py
A = int(input())
B = int(input())
C = int(input())
if A + B + C != 180:
  print("Error")
elif A == B == C:
  print("Equilateral")
elif A == B or B == C or A == C:
  print("Isosceles")
else:
  print("Scalene")
```
