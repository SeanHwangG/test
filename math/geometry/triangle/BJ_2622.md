> Question

* find the number of distinct triangle with total length of n

```txt
Input: 9
Output: 3
```

{% endtab %}
{% tab title='BJ_2622.py' %}

```py
ans = 0
for c in range(int(input())):
  if 2 * c < n <= 3 * c:
    ans += c - (n - c - 1) // 2
print(ans)
```
