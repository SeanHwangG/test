> Question

* Given L (consecutive days), P (maximum days in L), V (vacations)

```txt
Input:
5 8 20
5 8 17
0 0 0

Output:
Case 1: 14
Case 2: 11
```

> Solution

* ans = V // P * L
* V % P must be smaller than L (constraint)

{% endtab %}
{% tab title='BJ_4796.py' %}

```py
for i in range(1,999):
  L, P, V = map(int, input().split())
  if L == 0: break
  print(f'Case {i}: {V // P * L + min(V % P, L)}')
```
