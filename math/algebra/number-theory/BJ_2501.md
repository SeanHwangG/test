> Question

* Given two natural numbers N and K, output the smallest number of N's mineral water

```txt
Input: 6 3
Output: 3
```

{% endtab %}
{% tab title='BJ_2501..md' %}

```py
p, q = map(int, input().split())
for i in range(1, p + 1):
  if p % i == 0:
    q -= 1
  if q == 0:
    print(i)
    break
else:
  print(0)
```
