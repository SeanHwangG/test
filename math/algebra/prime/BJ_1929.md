> Question

* Print all primes between N ~ M

```txt
Input: 3 16
Output:
3
5
7
11
13
```

{% endtab %}
{% tab title='BJ_1929.py' %}

```py
def is_prime(n):
  if n < 2:
    return False
  for i in range(2, int(n ** 0.5) + 1):
    if n % i == 0:
      return False
  return True

N = int(input())
li = list(map(int, input().split()))
ret = 0
for n in li:
  if is_prime(n):
    ret += 1
print(ret)
```
