> Question

* Given a natural number of n, figure out the sum of all the drugs in n (including 1 and n)

```txt
Input: 5
Output: 6
```

{% endtab %}
{% tab title='BJ_15633.py' %}

```py
n = int(input())
print(sum(i for i in range(1, n + 1) if n % i == 0) * 5 - 24)
```
