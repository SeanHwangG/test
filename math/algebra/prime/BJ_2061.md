> Question

* Print GOOD if prime else BAD

```txt
Input: 143 11
Output: GOOD  # 2, 3 ... 10 doesn't divide 143
```

{% endtab %}
{% tab title='BJ_2061.py' %}

```py
p, k = map(int,input().split())
for i in range(2, k):
  if p % i == 0:
    print("BAD", i)
    break
else:
  print("GOOD")
```
