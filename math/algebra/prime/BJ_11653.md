> Question

* When given an integer N, write a small factorization program

```txt
Input: 72

Output:
2
2
2
3
3
```

{% endtab %}
{% tab title='BJ_11653.py' %}

```py
n = int(input())
for i in range(2, n + 1):
  while n % i == 0:
    n //= i
    print(i)
```
