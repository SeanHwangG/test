> Question

* Find number of primes

```txt
Input:
4
1 3 5 7

Output: 3
```

{% endtab %}
{% tab title='BJ_1978.py' %}

```py
def is_prime(n):
  if n < 2:
    return False
  for i in range(2, int(n ** 0.5) + 1):
    if n % i == 0:
      return False
  return True
N = int(input())
arr = list(map(int, input().split()))
ret = 0
for n in arr:
  if is_prime(n):
    ret += 1
print(ret)
```
