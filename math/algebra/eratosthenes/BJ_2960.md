> Question

* When given N and K, find the number of K-th erases in Eratonetenes's sieve

```txt
Input: 10 7
Output: 9
```

{% endtab %}
{% tab title='BJ_2960.py' %}

```py
N, K = map(int, input().split())
is_prime = [False, False] + [True] * (N - 1)
for i in range(2, N + 1):
  if is_prime[i]:
    for j in range(i, len(is_prime), i):
      if is_prime[j]:
        is_prime[j] = False
        K -= 1
      if K == 0:
        print(j)
        break
```
