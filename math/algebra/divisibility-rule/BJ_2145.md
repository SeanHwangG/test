> Question

* While number is single digits replace number to sum of digits

```txt
Input:
673
51
1000
99
0

Output:
7
6
1
9
```

{% endtab %}
{% tab title='BJ_2145.py' %}

```py
while 1:
  n = int(input())
  if n == 0:
    break
  print(9 if n % 9 == 0 else n % 9)
```
