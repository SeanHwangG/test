> Question

* Convert into oct number
  * \- : 0, \ : 1, ( : 2, @ : 3, ? : 4, > : 5, & : 6, %: 7, /: -1

```txt
Input:
(@&
?/--
/(\
?
#

Output:
158
1984
-47
4
```

{% endtab %}
{% tab title='BJ_1864.py' %}

```py
ch2int = {'-':0, '\\':1, '(':2, '@':3, '?':4, '>':5, '&':6, '%':7, '/':-1}
while True:
  string = input()
  if string == '#':
    break
  ret = 0
  for ch in string:
    ret *= 8
    ret += ch2int[ch]
  print(ret)
```
