> Question

* calculate the number of 1's in their binary representation and return them as an array

```txt
Input: n = 5
Output: [0,1,1,2,1,2]
```

{% endtab %}
{% tab title='LC_338.py' %}

```py
def countBits(self, num) -> List[int]:
  ret = [0] * (num + 1)
  for i in range(num + 1):
    ret[i] = ret[i // 2] + i % 2
  return ret
```
