> Question

* Find all 1's position in binary

```txt
Input:
1
13

Output: 0 2 3
```

{% endtab %}
{% tab title='BJ_3460.py' %}

```py
n = int(input())
for _ in range(n):
  x = int(input())
  n = 0
  while x != 0:
    if x & 1:
      print(n, end=' ')
    n += 1
    x //= 2
  print()
```
