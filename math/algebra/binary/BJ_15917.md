> Question

* Determine if a number can be represented by the power of 2?

```txt
Input:
10
1
2
7
4
14
32
33
34
35
36

Output:
1
1
0
1
0
1
0
0
0
0
```

{% endtab %}
{% tab title='BJ_15917.py' %}

```py
import sys
input = sys.stdin.readline
N = int(input())
for _ in range(N):
  a = int(input())
  if a&(-a) != a:
    print('0')
  else:
    print('1')
```
