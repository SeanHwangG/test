> Question

* Find number of 1 in binary

```txt
Input: n = 00000000000000000000000000001011
Output: 3
```

{% endtab %}
{% tab title='LC_191.cpp' %}

```cpp
int hammingWeight(uint32_t n) {
  return __builtin_popcount(n);
}
```

{% endtab %}
{% tab title='LC_191.py' %}

```py
def hammingWeight(self, n: int) -> int:
  return bin(n).count('1')
```
