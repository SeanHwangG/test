> Question

* Print decimal into B's number

```txt
Input: 60466175 36
Output: ZZZZZ
```

{% endtab %}
{% tab title='BJ_11005.py' %}

```py
a, b = map(int, input().split())
conv = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
ret = ""
while a != 0:
  ret = conv[a % b] + ret
  a //= b
print(ret)
```
