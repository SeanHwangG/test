> Question

* two 5 numbers are given
* print if none of them match

```txt
Input:
1 1 0 1 0
0 0 1 0 1

Output: Y
```

{% endtab %}
{% tab title='BJ_10395.py' %}

```py
a1, b1, c1, d1, e1 = map(int, input().split())
a2, b2, c2, d2, e2 = map(int, input().split())
if a1 ^ a2 and b1 ^ b2 and c1 ^ c2 and d1 ^ d2 and e1 & e2:
  print("Y")
else:
  print("N")
```
