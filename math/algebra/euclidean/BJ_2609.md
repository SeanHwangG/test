> Question

* Find GCD and LCM of each pair

```txt
Input: 24 18
Output:
6
72
```

{% endtab %}
{% tab title='BJ_2609.py' %}

```py
def GCD(a, b):
   while(b):
     a, b = b, a % b
   return a
def LCM(a, b):
  return a * b // GCD(a, b)
a, b = map(int, input().split())
print(GCD(a, b), LCM(a, b))
```
