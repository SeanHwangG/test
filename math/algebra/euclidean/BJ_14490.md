> Question

* Print in irreducible fraction

```txt
Input: 100:10
Output: 10:1
```

{% endtab %}
{% tab title='BJ_14490.py' %}

```py
import math
a, b = map(int,input().split(':'))
c = math.gcd(a,b)
print(a // c, b // c, sep = ':')
```
