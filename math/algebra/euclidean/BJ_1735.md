> Question

* Print sum of two fraction as irreducible fraction

```txt
Input:
2 7
3 5

Output: 31 35
```

{% endtab %}
{% tab title='BJ_1735.py' %}

```py
import math
a, b = map(int,input().split())
c, d = map(int,input().split())
m = a*d+b*c
n = b*d
g = math.gcd(m,n)
print(m//g,n//g)
```
