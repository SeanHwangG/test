> Question

* Find Largest common mulitple of two number

```txt
Input:
3
15 21
33 22
9 10

Output:
105
66
90
```

{% endtab %}
{% tab title='BJ_5347.py' %}

```py
def GCD(a, b):
  while(b):
    a, b = b, a % b
  return a
def LCM(a, b):
  return a * b // GCD(a, b)
for _ in range(int(input())):
  a, b = map(int, input().split())
  print(LCM(a, b))
```
