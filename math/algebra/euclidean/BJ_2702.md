> Question

* Find LCM, GCD of two integers

```txt
Input:
3
5 10
7 23
42 56

Output:
10 5
161 1
168 14
```

{% endtab %}
{% tab title='BJ_2702.py' %}

```py
def GCD(x, y):
   while y :
     x, y = y, x % y
   return x
def LCM(a, b):
  return a * b // GCD(a, b)
for _ in range(int(input())):
  a, b = map(int, input().split())
  print(LCM(a, b), GCD(a, b))
```
