> Question

* Find LCM of each pairs

```txt
Input:
3
1 45000
6 10
13 17

Output:
45000
30
221
```

{% endtab %}
{% tab title='BJ_1934.py' %}

```py
def GCD(a, b):
  while(b):
    a, b = b, a % b
  return a
def LCM(a, b):
  return a * b // GCD(a, b)
for _ in range(int(input())):
  a, b = map(int, input().split())
  print(LCM(a, b))
```
