> Question

* N natural numbers are given. find all the pledges of this natural number

```txt
Input:
2
75 125

Output:
1
5
25
```

{% endtab %}
{% tab title='BJ_5618.py' %}

```py
def GCD(x, y):
  while(y):
    x, y = y, x % y
  return x

input()
s = list(map(int, input().split()))
g = GCD(s[0], GCD(s[1], s[-1]))
for i in range(1, (g // 2) + 1):
  if g % i == 0:
    print(i)
print(g)
```
