> Question

* Print Nth fibonacci number (N < 1,000,000,000,000,000,000)

```txt
Input: 1000
Output: 517691607
```

{% endtab %}
{% tab title='BJ_11444.py' %}

```py
def mul(a,b):
  return [[sum(i * j for i, j in zip(x, y)) % 1000000007 for y in b] for x in a]
def power(a,n):
  if n > 1:
    x = power(a, n//2)
    x = mul(x,x)
    if n % 2 != 0:
      return mul(x,a)
    else:
      return(x)
  else:
    return(a)

print(power([[1,1],[1,0]],int(input()))[1][0])
```
