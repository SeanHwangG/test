> Question

* Your script should output the following valid phone numbers:

```txt
Input:

987-123-4567
123 456 7890
(123) 456-7890

Output:
987-123-4567
(123) 456-7890
```

```sh
grep -e "^\([0-9]\{3\}-\|([0-9]\{3\}) \)[0-9]\{3\}-[0-9]\{4\}$" telephone.txt
```
