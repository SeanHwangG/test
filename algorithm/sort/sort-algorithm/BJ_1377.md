> Question

* Find the number of iteration for bubble sort

```txt
Input:
5
10
1
5
2
3

Output: 3
```

{% endtab %}
{% tab title='BJ_1377.py' %}

```py
L = [(int(input()), i) for i in range(int(input()))]
L.sort()
m = 0
for i in range(len(L)):
  m = max(L[i][1] - i, m)
print(m + 1)
```
