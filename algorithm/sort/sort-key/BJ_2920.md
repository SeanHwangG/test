> Question

* Print if number is ascending / descending or mixed

```txt
Input: 1 2 3 4 5 6 7 8
Output: ascending
```

{% endtab %}
{% tab title='BJ_2920.py' %}

```py
li = list(map(int, input().split()))
if li == sorted(li):
  print("ascending")
elif li == sorted(li, reverse=True):
  print("descending")
else:
  print("mixed")
```

read s;
