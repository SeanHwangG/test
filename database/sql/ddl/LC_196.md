> Question

| Id  | Email            |
| --- | ---------------- |
| 1   | john@example.com |
| 2   | bob@example.com  |
| 3   | john@example.com |

* delete all duplicate email entries in a table named Person, keeping only unique emails based on its smallest Id

| Id  | Email            |
| --- | ---------------- |
| 1   | john@example.com |
| 2   | bob@example.com  |

{% endtab %}
{% tab title='LC_196.py' %}

```sql
DELETE p1 FROM Person p1, Person p2 WHERE p2.Email = p1.Email AND p2.Id < p1.Id;
```
