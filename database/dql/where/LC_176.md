> Question

| Id  | Salary |
| --- | ------ |
| 1   | 100    |
| 2   | 200    |
| 3   | 300    |

* Write a SQL query to get the second highest salary from the Employee table

| SecondHighestSalary |
| ------------------- |
| 200                 |

{% endtab %}
{% tab title='LC_176.sql' %}

```sql
SELECT MAX(Salary) SecondHighestSalary FROM Employee
  WHERE Salary != (SELECT MAX(Salary) FROM Employee);
```
