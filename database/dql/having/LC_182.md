* Write a SQL query to find all duplicate emails in a table named Person

| Email   |
| ------- |
| a@b.com |

{% endtab %}
{% tab title='LC_182.sql' %}

```sql
SELECT email FROM Person
  GROUP BY email HAVING count(*) > 1;
```
