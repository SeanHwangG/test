* query all the consecutive available seats order by the seat_id

| seat_id |
| ------- |
| 3       |
| 4       |
| 5       |

{% endtab %}
{% tab title='LC_603.sql' %}

```sql
SELECT DISTINCT a.seat_id FROM cinema a
  JOIN cinema b ON abs(a.seat_id - b.seat_id) = 1 AND a.free=true AND b.free=true
  ORDER BY a.seat_id;
```
