> Question

* For each directed graph, check for negative cycle

```txt
Input:
2
3 3 1
1 2 2
1 3 4
2 3 1
3 1 3
3 2 1
1 2 3
2 3 4
3 1 8

Output:
NO
YES
```

{% endtab %}
{% tab title='BJ_1865.py' %}

```py
for _ in range(int(input())):
  n, m, w = map(int, input().split())
  A = []
  for _ in range(m):
    s, e, t = map(int, input().split())
    A += [(s, e, t), (e, s, t)]
  for _ in range(w):
    s, e, t = map(int, input().split())
    A += [(s, e, -t)]

  dp = [0] * (n + 1)
  cycle = False
  for i in range(n):
    for s, e, t in A:
      if dp[e] > dp[s] + t:
        dp[e] = dp[s] + t
        if i == n - 1:
          cycle = True
  if cycle:
    print("YES")
  else:
    print("NO")
```
