> Question

* print sum when two smallest elements keep added and pushed it back

```txt
Input:
3
10
20
40

Output: 100  # (10 + 20) + (30 + 40)
```

{% endtab %}
{% tab title='BJ_1715.py' %}

```py
from heapq import *
l = [int(input())for i in range(int(input()))]
heapify(l)
s = 0
while len(l) > 1:
  t = heappop(l) + heappop(l)
  s += t
  heappush(l, t)
print(s)
```
