# X86

* dword ptr part is called a size directive

## Data

* qword: 8 byte

## Operation

* mov `a` `b`: move `b` to `a`
* add `a` 1: add 1 to `a`
  * Only valid for register
* incl `a`: increment `a` by 1

# counter += 1
