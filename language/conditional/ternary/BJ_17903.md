> Question

* Given a, b
* if a is greater than 8 print satisfactory else print unsatisfactory

```txt
Input:
5 3
-1 2 3
-1 -2 3
1 -2 3
1 -2 -3
1 2 -3

Output: unsatisfactory
```

{% endtab %}
{% tab title='BJ_17903.py' %}

```py
a, b= map(int, input().split())
print("satisfactory" if a >= 8 else "unsatisfactory")
```
