> Question

* Print number of integers where difference of each consequent digit are equal

```txt
Input: 110
Output: 99
```

{% endtab %}
{% tab title='BJ_1065.py' %}

```py
N = int(input())
print(sum(x // 100 + x % 10 == x // 10 % 10 * 2 or x < 100 for x in range(1, N + 1)))
```

{% endtab %}
{% tab title='BJ_1065.sh' %}

```sh
read num
cnt=0

for ((i=1; i<=$num; i++)); do
  if [ $i -lt 100 ]; then
    ((cnt++))
  else
    a=${i:0:1}
    b=${i:1:1}
    c=${i:2:1}

    if [ $(($a - $b)) -eq $(($b - $c)) ]; then
      ((cnt++))
    fi

  fi
done

echo $cnt
```
