> Question

* When given two positive integers, outputs both integers between them

```txt
Input: 8 14
Output:
5
9 10 11 12 13
```

{% endtab %}
{% tab title='BJ_10093.cpp' %}

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
  long long a, b;
  cin >> a >> b;
  if (a == b)cout << "0\n";
  else cout << abs(b - a)-1 << '\n';
  for (long long i = min(a,b) + 1; i < max(a, b); i++) cout << i << ' ';
}
```

{% endtab %}
{% tab title='BJ_10093.py' %}

```py
a, b = map(int, input().split())
n1 = min(a, b)
n2 = max(a, b)
print(n2 - n1 - 1 if n2 > n1 else 0)
for i in range(n1+1, n2):
  print(i, end = ' ')
```
