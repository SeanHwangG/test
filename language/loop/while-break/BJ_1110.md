> Question

* Starts from 26. 2+6 = 8. 6+8 = 14. 8+4 = 12. 4+2 = 6
* print length of cycle

```txt
Input: 26
Output: 4
```

{% endtab %}
{% tab title='BJ_1110.py' %}

```py
n = int(input())
m = n
i = 0
while True:
  m = m % 10 * 10 + (m % 10 + m // 10) % 10
  i += 1
  if m == n:
    print(i)
    break
```

{% tab title='BJ_1110.sh' %}

```sh
read init

temp=init
count=0

while [ $temp -ne $init ]; do
  let front=$((temp%10))
  let rear=$((temp%10+temp/10))
  rear=$((rear%10))
  temp=$((front*10+rear))
  count=$((count+1))
done

echo "$count"
```
