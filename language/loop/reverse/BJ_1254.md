> Question

* Print shortest palindrome length that can be made

```txt
Input: abab
Output: 5
```

{% endtab %}
{% tab title='BJ_1254.py' %}

```py
s = input()
i, b =0, s
while b != b[::-1]:
  b = s + s[i::-1]
  i += 1
print(len(b))
```
