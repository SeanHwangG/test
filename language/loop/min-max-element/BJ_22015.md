> Question

* Given A, B, C, print biggest element * 3 - sum of all elements

```txt
Input: 4 6 9
Output: 8
```

{% endtab %}
{% tab title='BJ_22015.py' %}

```py
A, B, C = map(int, input().split())
print(3 * max(A, B, C) - A - B - C)
```
