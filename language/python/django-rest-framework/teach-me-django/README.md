# Teach Me Django

This project contains a Django Rest API with a React frontend
Users are able to signup and login to their account
Teeachers are able to create quizes
Students are able to answer those quizes and view their results

```sh
# 1. Backend development workflow
virtualenv venv
pip install -r requirements.txt
python manage.py runserver

# 2. Frontend development workflow
npm i
npm start

# 3. For deploying
npm run build
```
